# 拉普拉斯 - 个人概述

   拉普拉斯 (Laplace,Pierre-Simon,marquisde)，法国著名数学家和天文学家，拉普拉斯是天体力学的主要奠基人，是天体演化学的创立者之一，是分析概率论的创始人，是应用数学的先躯。拉普拉斯用数学方法证明了行星的轨道大小只有周期性变化，这就是著名拉普拉斯的定理。他发表的天文学、数学和物理学的论文有 270 多篇，专著合计有 4006 多页。其中最有代表性的专著有《天体力学》、《宇宙体系论》和《概率分析理论》。1796 年，他发表《宇宙体系论》。因研究太阳系稳定性的动力学问题被誉为法国的牛顿和天体力学之父。
